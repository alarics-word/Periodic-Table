Periodic Table Temperature Phase Simulator

An interactive Periodic Table Web App that simulates how chemical elements change state based on temperature.
As the temperature scale changes, each element dynamically updates its physical state (Solid, Liquid, or Gas) in real time.

How It Works

Each element contains:

Melting point
Boiling point

The app compares the current temperature against those values:

If temperature < melting point → Solid
If temperature ≥ melting point and < boiling point → Liquid
If temperature ≥ boiling point → Gas

The UI updates automatically whenever the slider changes.
